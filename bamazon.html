MySQL Project: Bamazon

Bamazon is a MySQL assignment given in my coding bootcamp. Since the assignment is displayed in the bash terminal, I will be providing screenshots and explanations below. Thank you!

Bamazon Customer

Screenshot Explanation

The images on the left display the customer prompt of Bamazon. Image 1 shows the inventory inside of the database. Following the inventory, the prompt starts up, showing that there are no errors upon running the document in the terminal. Next, image 2 displays what product the customer ordered, the quantity of the product ordered, and the successful transaction. Image 3 displays the updated quantity of the item purchased and the prompt reset.

Screenshot Explanation

The images on the right display the manager prompt of Bamazon. The top set of images displays adding new products to the inventory, while the bottom images display updating the quantity of existing products. Top set - Image 1 shows the inventory list followed by a prompt that asks the user to choose whether they want to update a product's quantity or add a new product. Choosing to add an item - Image 2 shows the product that the user wants to add to the inventory along with the quantity, department, price, and confirmation that the product has been added. Image 3 shows that the newly added product in the inventory and resets the prompt for the user to add another product or update the quantity of an existing product.

The bottom images - Image 1 displays the current product's quantity you want to restock. Image 2 showsthe product of choice, the restock quantity, and confirmation that the product has been restocked. Image 3 displays the updated quantity available and the prompt reset.
